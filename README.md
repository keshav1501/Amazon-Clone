# Amazon Clone

A clean and modern Amazon homepage clone built using HTML5 and CSS3. This project was created to improve front-end development skills by recreating the layout and styling of Amazon's homepage.

## Features

- Responsive navigation bar
- Search bar with category selector
- Delivery location section
- Country selector
- Account, Orders, and Cart sections
- Hero banner
- Product category cards
- Multi-column footer
- Hover effects
- Font Awesome icons

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Font Awesome
