# 🛒 Amazon Clone

A simple Amazon homepage clone built using **HTML5** and **CSS3**. This project replicates the layout of Amazon's homepage to practice front-end web development skills.

## 🚀 Features

- Responsive-inspired layout
- Navigation bar
- Search bar
- Hero section
- Product cards
- Footer
- Font Awesome icons

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
